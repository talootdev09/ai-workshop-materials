# Day 1: LLM APIs & Code Basics

## Overview

Day 1 focuses on building a solid foundation in LLM APIs and Python programming.

## Exercises

### Exercise 1: First API Call
**File**: `exercises/01_first_api_call_starter.py`

**Learning Goals**:
- Load environment variables
- Create OpenAI client
- Make your first API call
- Extract and display response

**Time**: 20 minutes

### Exercise 2: Prompt Engineering
**File**: `exercises/02_prompt_engineering_starter.py`

**Learning Goals**:
- Understand system messages
- Use few-shot prompting
- Control creativity with temperature
- Request structured outputs

**Time**: 30 minutes

### Exercise 3: Chatbot with Memory
**File**: `exercises/03_chatbot_memory_starter.py`

**Learning Goals**:
- Maintain conversation history
- Build a working chatbot
- Understand context in conversations

**Time**: 40 minutes

### Bonus: Streaming Responses
**File**: `bonus/04_streaming_bonus.py`

**Learning Goals**:
- Implement streaming responses
- Create better UX for long responses

**Time**: Optional (if ahead of schedule)

## How to Use

1. Start with exercises (in `exercises/` folder)
2. Try to complete them yourself
3. Check solutions (in `solutions/` folder) if stuck
4. Run bonus exercise if time permits

## Solutions

All solutions are in the `solutions/` folder. Compare your code with the solutions to see different approaches and best practices.
